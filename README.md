
## Next.js 前后端项目
> next.js+typescript+prisma+mysql

运行以下命令安装依赖：

```bash
npm install
# or
cnpm install
# or
yarn
# or
pnpm install
```

启动项目：
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

发布上线
```bash
npm run build
```

```bash
npm run start
```

##### prisma
> 数据库模型文件， `model` 对应每一个表
